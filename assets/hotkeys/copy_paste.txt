 Nusxalash, joylashtirish va umumiy Windows tugmalari
Ctrl + X
Tanlangan elementni kesib oling

Ctrl + C (yoki Ctrl + Insert)
Tanlangan elementni nusxalash

Ctrl + V (yoki Shift + Insert)
Tanlangan elementni joylashtiring

Ctrl + Z
Harakatni bekor qilish

Alt + Tab
Ochiq ilovalar o'rtasida almashish

Alt + F4
Faol elementni yoping yoki faol ilovadan chiqing

Windows belgisi + L
Kompyuteringizni qulflang

Windows belgisi + D
Ish stolini ko'rsatish va yashirish

F2
Tanlangan element nomini o'zgartiring

F3
File Explorer-da fayl yoki papkani qidiring

F4
Fayl Explorer-da manzillar paneli ro'yTagiga chizish belgisiini ko'rsatish

F5
Faol oynani yangilang

F6
Oynada yoki ish stolida ekran elementlarini aylantiring

F10
Faol ilovadagi Menyu panelini faollashtiring

Alt + F8
Enter ekranida parolingizni ko'rsating

Alt + Esc
Elementlarni ochilgan tartibda aylantiring

Alt + Tagiga chizish belgisi
Ushbu harf uchun buyruqni bajaring

Alt + Enter
Tanlangan element uchun xususiyatlarni ko'rsatish

Alt + Boʻsh joy(probel)
Faol oyna uchun yorliq menyusini oching

Alt + ←
Ortga qaytish

Alt + →
Olg'a yurish

Alt + Page Up
Bir ekran yuqoriga siljiting

Alt + Page Down
Bir ekranni pastga siljiting

Ctrl + F4
Faol hujjatni yoping (to'liq ekranli va bir vaqtning o'zida bir nechta hujjatlarni ochishga imkon beradigan ilovalarda)

Ctrl + A
Hujjat yoki oynadagi barcha elementlarni tanlang

Ctrl + D (yoki Delete)
Tanlangan elementni o'chiring va uni axlat qutisiga o'tkazing

Ctrl + R (yoki F5)
Faol oynani yangilang

Ctrl + Y
Harakatni qayta bajaring

Ctrl + →
Kursorni keyingi so'zning boshiga olib boring

Ctrl + ←
Kursorni oldingi so'zning boshiga olib boring

Ctrl + ↓
Kursorni keyingi paragrafning boshiga olib boring

Ctrl + ↑
Kursorni oldingi paragrafning boshiga olib boring

Ctrl + Alt + Tab
Barcha ochiq ilovalar o'rtasida almashish uchun Strelka tugmalaridan foydalaning

Ctrl + Alt + Shift + Strelka tugmalari
Guruh yoki kafel Ishga tushirish menyusida diqqat markazida bo'lsa, uni belgilangan yo'nalishda harakatlantiring

Ctrl + Strelka tugmasi (elementga o'tish uchun) + bo'sh joy
Oynada yoki ish stolida bir nechta alohida elementlarni tanlang

Ctrl + Shift Strelka tugmasi bilan
Matn blokini tanlang

Ctrl + Esc
Start menyuni oching

Ctrl + Shift + Esc
Vazifa menejerini oching

Ctrl + Shift
Bir nechta klaviatura tartibi mavjud bo'lganda klaviatura tartibini almashtiring

Ctrl + Boʻsh joy
Xitoycha kiritish usuli muharririni (IME) yoqing yoki oʻchiring

Shift + F10
Tanlangan element uchun yorliq menyusini ko'rsating

<- ^ -> Har qanday Strelka tugmasi bilan siljiting


Shift + Strelka
Oynada yoki ish stolida bir nechta elementni tanlang yoki hujjatdagi matnni tanlang


Shift + Delete Tanlangan elementni avval axlat qutisiga o'tkazmasdan o'chiring

→
O'ngdagi keyingi menyuni oching yoki pastki menyuni oching

←
Chapdagi keyingi menyuni oching yoki pastki menyuni yoping

Esc
Joriy vazifani to'xtating yoki qoldiring
